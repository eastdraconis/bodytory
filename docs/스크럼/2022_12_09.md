## 디테일

- 이미지 저장할 떄 이름 바꾸기 : 동룡
- 만 나이 계산하기 : 동룡
- 병원 이름 자르기 : 동룡
- 맵에 추가 기능 연결 하기 : 호성
- 리스트서치 처음에 스켈레톤 뜨는거 체크 : 경원
- 맵에서 내가 등록한 병원은 다르게 표시 : 경원
- 진료과목 없는거는 (기타로 연결 or 진료과목 라벨 없애기) : 경원
- 메일 전송, 포스터? 디자인 : 소희
